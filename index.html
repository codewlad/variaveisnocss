Usando variáveis no CSS

Por que usar?

O uso de variáveis facilita bastante quando é preciso alterar a cor de vários elementos da página, mas sem mudar manualmente cada um deles.

Como usar?

Primeiro, criamos uma pseudoclasse :root, e definimos uma váriável que vai armazenar a cor desejada.

No exemplo abaixo, criamos uma variável que vai armazenar a cor:

:root {
   --main-h2-color: #993299;
}

Agora, basta usar a variável criada para todos os elementos que terão a mesma cor, usando o var(--nome-da-variavel):

h2 {
   var(--main-h2-color);
}

Quando quiser alterar a cor de todos os elementos de uma só vez, basta trocar a cor da variável e pronto!
